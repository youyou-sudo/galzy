# 游戏条目 API

获取指定游戏的详细信息与文件列表。

---

## 基本信息

- **接口地址**：`/api/game`
- **请求方法**：`GET`
- **请求类型**：`application/json`
- **返回类型**：`application/json`

---

## 请求参数
**vid**
-  类型：`string`
-  必填：✅
-  说明：游戏的 VNDB ID，例如 `v54`

## 类型

```typescript
export interface ApiResponse {
  code: number
  message: string
  data: GameData
}

export interface GameData {
  vid: string
  title: string
  alias?: string | null
  description?: string | null
  filelist: FileItem[]
}

export interface FileItem {
  id: string
  name: string
  type: 'file' | 'folder'
  size?: string
  format?: string
  children?: FileItem[]
}

```

## 错误响应示例

- 游戏 ID 缺失或无效
  ```json
  {
    "code": "400",
    "message": "喵喵什么都不知道喵，请提供正确的游戏 ID 喵～"
  }
  ```

- 游戏无文件
  ```json
  {
    "code": "400",
    "message": "喵喵什么都不知道喵，此游戏暂时未上传文件喵～"
  }
  ```

## 状态码说明
- `200`
  - 请求成功，返回游戏详情
- `400`
  - 请求参数错误或未找到游戏
